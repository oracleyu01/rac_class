▣ 예제5. RAC 환경에서 일관된 data 를 볼 수 있는 기술은 무엇인가 ?

 rac 환경에서는 어느 노드로 접속하든지 간에 항상 일관된 데이터를 봅니다.

그림 : https://cafe.daum.net/oracleoracle/SoJX/6

설명:  rac 환경에서는 내가 어느 노드로 접속하든지 간에 항상 맨 마지막에 commit 된 최신 데이터를 일관되게 조회 할 수 있습니다.

- 최신 데이터가 어느 노드에 있는지 어떻게 알 수 가 있는가 ?

GRD(Global  resource  directory) 에 최신 데이터가 어느 노드에 있다라고 위치정보가 있기 때문입니다. 

GRD 는 각각의 노드의 shared pool 에 존재합니다. 

예를 들면 emp 테이블의 최신 데이터의 위치정보는 1번 노드의 GRD 에 있고 dept 테이블의 최신 데이터의 위치 정보는 2번 노드의 GRD 에 있다. 


그림 설명 : https://cafe.daum.net/oracleoracle/SoJX/7

설명:   
emp 테이블에 대한 최신 데이터가 어디 있는지 그 위치정보가 1번 노드의 GRD 에 있다고 한다면 2번 노드로 접속한 유져가 EMP 테이블의 DATA 를
SELECT 할 때 그 위치 정보를 알기 위해서 1번 노드의 grd 에 물어봅니다. 그리고 위치정보를 알아내고 그리고 데이터를 select 합니다.
