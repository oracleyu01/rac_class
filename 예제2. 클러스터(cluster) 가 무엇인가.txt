▣ 예제2.  클러스터(cluster) 가 무엇인가 ?  

그림 : I-6 페이지 그림 

집단이라는 뜻인데 여러대의 노드(서버)들의 집단을 마치 하나의 서버처럼 보이게 하는 소프트웨어를 cluster 라고 합니다. 

클러스터 소프트웨어가  오라클 클러스터 소프트웨어를 설치를 해야 rac 를 설치할 수 있습니다.

HP , IBM 에 클러스터 소프트웨어가 있는데 오라클의 클러스터 소프트웨어를 설치해야 오라클 RAC 를 쓸 수 있습니다. 

RAC 설치할 때 순서:

 1. grid 설치( cluster + ASM )
 2. DB 설치

grid 설치 ---> oracle db 소프트웨어 설치 ----> dbca 로 db 생성
